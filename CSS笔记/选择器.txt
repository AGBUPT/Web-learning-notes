类选择器

ID选择器

通用选择器（应用于所有元素）
* {
  /* style content */
}

伪类
  伪类用来表示动态事件
  :link 未访问的超链接
  :visited 访问过的超链接
  :hover 鼠标指针悬停在的元素（鼠标）
  :focus 当前获得焦点的元素(键盘）
  :active 用户正在单击该元素(如果一直按着鼠标左键，则此样式持续）

  ·其中，:link 和 :visited 只能用于<a>

优先级
  · !important 优先级最高，其余的可以记作：
  · style > id > class > 标签
      一般来讲笼统记住这个式子就够了
  · 具体权值
      style=1000，id=100，class=属性=伪类=10，标签= 1
     （如有多个class、属性、伪类或标签，则权值叠加）
  · 忽略伪元素的权值

继承
  大多数情况下，文本和字体属性可以继承

最佳实践
  · 使每个选择器的针对性尽可能低
  · 不要使用style属性和!important，这两个让用户很难自定义自己的规则，也不便于网站维护
